# Hey, I'm Arsalan

Software engineer at NatureDots, building products for environmental and aquaculture monitoring — interactive maps, real-time sensor data, geospatial visualization.

Outside of work I'm mostly in Go, working on backend systems, distributed architecture, and developer tooling. Currently building **DevDeploy**, a self-hosted deployment platform that takes a Git URL and handles the rest — clone, build via Railpack, containerize, deploy, track. The stack is Go, RabbitMQ, Redis, PostgreSQL, and Docker, with Cloudflare Tunnel handling external access without exposing ports.

I like problems where the system design actually matters: message queues, async job processing, deployment automation, data-intensive applications. I've built a document Q&A platform with Redis Streams and vector search, a distributed URL shortener with Keycloak auth, an AI-powered interview platform, and a handful of full-stack apps along the way.

Most of my professional work lives in private repos. What's here is personal and side projects.
